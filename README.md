# Your Portfolio Website

A single-file portfolio (`index.html`). No build step, no dependencies, no cost.
Just edit and host.

## 1. Personalize it

Open `index.html` in any text editor and search for `EDIT:` — every spot you
should change is marked with a comment like:

```html
<!-- ===== EDIT: your name ===== -->
```

Update these:
- Your name, role, and intro (hero section)
- About text and quick facts
- Skills chips
- Projects — title, description, tech tags, demo/source links
- Experience and education
- Contact links: email, GitHub, LinkedIn

Tip: replace `ashna.mercy@example.com` with your real email and
`yourusername` with your real GitHub/LinkedIn handles.

## 2. Preview locally

Just double-click `index.html` to open it in your browser. That's it.

## 3. Host it for free

### Option A — GitHub Pages (recommended, gives you a permanent URL)
1. Create a free account at https://github.com
2. Create a new repository named `yourusername.github.io`
3. Upload `index.html` to it (drag-and-drop in the GitHub web UI works)
4. Go to **Settings → Pages**, set source to the `main` branch
5. Your site is live at `https://yourusername.github.io` within a minute or two

### Option B — Netlify (drag-and-drop, super fast)
1. Sign up free at https://netlify.com
2. Go to the "Sites" area and drag the folder containing `index.html` onto the page
3. You get an instant live URL (you can rename it in site settings)

### Option C — Cloudflare Pages / Vercel
Both have free tiers and work the same way: connect a GitHub repo or upload the file.

## 4. Custom domain (optional, small cost)
Hosting is free. If you later want `yourname.com`, buy a domain (~$10/yr) and
point it at GitHub Pages or Netlify in their dashboard settings.
